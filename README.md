# Movie Magic

## Description
Movie Magic is a movie database application that allows users to view the latest trending movies, search for titles in our vast collection, and add favorites for easy access.

## Installation
To get started with Movie Magic, follow these steps:

1. Install the RPM package.
2. Run the application using the command:
npm run dev

## Usage
- Browse trending movies.
- Use the search feature to find specific films.
- Add your favorite movies to your personal list for quick access.
